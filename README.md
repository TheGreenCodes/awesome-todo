# awesome-todo

> A To-Do application highlighting vue unit testing


## Series playbook:

https://thegreencodes.com/series/vue-unit-testing



## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
